# タイピングゲーム「Alphabet Master」仕様書

## 1. 概要
- **ターゲット**: 小学4年生
- **目的**: アルファベット（大文字・小文字）の配置と、正しい運指（タッチタイピング）の習得。
- **プラットフォーム**: Webブラウザ（PCキーボード入力前提）
  - ※スマートフォンからアクセスされた場合は、「PCのキーボードを使って遊んでね！」という警告画面を表示し、ゲームプレイを制限する。
- **技術スタック**: HTML5, CSS3, JavaScript (すべてVanilla)

## 2. ゲームモード
タイトル画面で以下の2つのモードを選択可能。初心者が迷わないよう、小文字モードのボタンには「はじめての人はこちら」「おすすめ！」といった推奨導線を配置する。
1. **小文字モード**: 画面に小文字が表示され、対応するキーを入力する。（タッチタイピングの基本習得）
2. **大文字モード（シフト入力練習）**: 画面に大文字が表示され、「Shiftキー + 対応するキー」を入力する。（Shiftキーを使った大文字入力の練習）

## 3. ゲームシステム・ルール

### 3.1. 基本ルール
- **制限時間**: 60秒（1分間）。タイムアップでゲーム終了。
- **入力フロー**: 画面中央に出題されるアルファベットに対し、正しいキーを入力する。正解すると次の問題が表示される。

### 3.2. レベルアップ（文字数増加）とペナルティ
初心者が挫折しないよう、段階的な難易度設計とペナルティ緩和を行う。
- **初期状態**: 1文字ずつ出題。
- **レベルアップ（コンボ）**: 連続正解（コンボ）が一定数（例: 10コンボごと）に達するとレベルが上がり、一度に出題される文字数が最大5文字まで増加する。
- **ミスの猶予（段階的ペナルティ）**:
  - ミスをしても即座にコンボやレベルはリセットされない。
  - 「ミスカウント」を保持し、3回ミスした時点でレベルが1段階（文字数が1つ）下がり、コンボもそのレベルの初期値に戻る。正解すればミスカウントはリセットされる。

### 3.3. スコアと記録
- **スコア計算**: 1文字正解につき 100pt ＋（現在のコンボ数 × 10pt）
- **ハイスコア保存**: 継続的なモチベーション維持のため、`localStorage` にハイスコアを保存する（ブラウザを閉じても記録保持）。

## 4. UI・演出（タッチタイピング支援とエフェクト）
小学4年生が夢中になれるよう、リッチでフィードバックの強いUI/UXを実装する。

### 4.1. ビジュアルデザイン
- **テーマ**: ダークモード基調に、鮮やかなネオンカラーや宝石のような配色（サイバーパンク風）で特別感を演出。
- **フォント**: 丸みがありつつもスタイリッシュなモダンフォント（Google Fonts等）。

### 4.2. タッチタイピング支援（仮想キーボード）
画面下部にQWERTY配列の仮想キーボード（左右両方のShiftキー配置）と、運指ガイド用の手のイラストを表示する。
- **キートップ**: すべて「大文字」表記。
- **ハイライト・先読みガイド**: 直近で入力すべきキーと該当する指を強くハイライトする。複数文字表示時は、次の1〜2文字先を薄くハイライトし、先読みを促進する。
- **大文字モードの運指ロジック**：「右手キーには左Shift、左手キーには右Shift」のルールに基づき、対象キーと適切なShiftキー・指を同時にハイライトする。

### 4.3. サウンド・エフェクト詳細
効果音（SE）は体験品質に直結するため必須で実装する。
- **正解時**: ポップ音。弾けるパーティクルアニメーション。「x5 COMBO!」等のテキストポップアップ。コンボ数に応じて正解音のピッチが上昇する。
- **ミス時**: ブザー音。画面シェイクと赤フラッシュで視覚的にフィードバック。
- **レベルアップ時**: 専用音。画面全体のフラッシュ、「LEVEL UP!」アニメーション。枠や背景エフェクトがより派手な配色（青→黄→赤→虹色など）に変化する。
- **リザルト画面**: 獲得スコアの高速カウントアップアニメーション。ハイスコア更新時はファンファーレ音と豪華な紙吹雪パーティクル。

## 5. 画面構成
1. **タイトル画面**: ゲームタイトル、モード選択ボタン
2. **プレイ画面**:
   - ヘッダー: 残り時間（大）、現在スコア、ハイスコア
   - 中央: 出題アルファベット
   - 画面下部: 仮想キーボード、運指ガイド（手のイラスト）
   - フッター/背景: 現在のコンボ数、レベル表示
3. **リザルト画面**: 最終スコア、最大コンボ数、ハイスコア、「もう一度遊ぶ」「タイトルへ戻る」ボタン
